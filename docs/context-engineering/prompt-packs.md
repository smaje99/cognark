# Prompt Packs

Prompt packs are task-oriented instruction sets for agents. They should be short, versioned, and scoped.

## Initial Families

- `workspace-bootstrap`: initialize or repair workspace structure.
- `artifact-authoring`: create or update document artifacts with valid frontmatter.
- `traceability`: inspect and maintain graph relationships.
- `backlog-runner`: implement prioritized requirements from the backlog.
- `decision-support`: draft ADRs and summarize tradeoffs.
- `meeting-followup`: derive actions, decisions, and context updates from meeting notes.

Prompt packs should reference retrieval policy and required verification steps.
